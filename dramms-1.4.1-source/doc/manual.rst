Registration Command
--------------------

.. raw:: html

    <br />
	
The main command of DRAMMS which registers two images and optionally performs further
analysis of the obtained deformation is named ``dramms``. The simplest use is: ::

  dramms --source  sourceimage.hdr   --target  targetimage.nii  
         --outimg  outimage.img      --outdef  outdef.nii.gz    


.. raw:: html

    <br />
	
**Supported File Formats:** NIfTI-1_ (recommended), `ANALYZE 7.5`_

**Supported Datatypes:** byte (unsigned char, uint8), int8, short, int16, uint16, float, float32, int32.

**Parameter Settings:** The default settings will give reasonable results in many cases.
We recommend a look at the :doc:`tutorials` page and :ref:`FAQ.8 <q8>`, :ref:`FAQ.9 <q9>`, :ref:`FAQ.10 <q10>` for task-specific uses.
If in a particular case neither the default nor the example settings give satisfactory results,
please refer to :ref:`FAQ.1 <q1>` for  the most important parameters to tune.

**Memory Settings:** The memory requirement is largely dependent on the image size and the specific task ---
see :ref:`SystemRequirements` for some general rules, and the :doc:`tutorials`
page for some task-specific examples. How to reserve the mount of memory can be found in :ref:`FAQ.4 <q4>`.
If ``dramms`` terminated due to the lack of available memory, please refer to :ref:`FAQ.5 <q5>` for how to reduce the memory consumption while still obtaining reasonable results.


**Other Usages:**  

- Multi-Modality Registration :ref:`FAQ.8 <q8>`  
- Registration with initialized deformation :ref:`FAQ.9 <q9>`  
- Registration with Cost-Function-Masking :ref:`FAQ.10 <q10>`

------------------

+---------------------------------------------------+-------------------+
| Calculate RAVENS maps                             | |dramms-ravens|   |
+---------------------------------------------------+-------------------+
| Invert/Smooth deformation, or                     | |dramms-defop|    |
| Print-out displacement at a voxel                 |                   |
+---------------------------------------------------+-------------------+
|Compose/Add/Subtract/Average two transformations   | |dramms-combine|  |
+---------------------------------------------------+-------------------+
